🍰 Sweet Cravings Bakery Website

A fully responsive bakery website with an interactive shopping cart and contact form, built using HTML, CSS, and JavaScript.

🚀 Live Features
🛍️ Add to Cart functionality
🧮 Auto quantity & total price calculation
🗑️ Remove items / Clear cart
💾 Cart data stored in LocalStorage
📱 Fully responsive design (mobile-friendly)
📩 Contact form with validation
🔄 Real-time error handling
🎯 Smooth scrolling navigation
🖥️ Tech Stack

HTML5 – Structure
CSS3 – Styling & Responsive Design
JavaScript (Vanilla JS) – Functionality & Logic
LocalStorage API – Data persistence

🛒 Cart Features Explained
Add items with "Add to Cart"
Automatically increases quantity if item exists
Displays:
Product name
Price
Quantity
Updates total price dynamically
Data persists even after refresh using LocalStorage

📩 Contact Form Validation
Name → Required
Email → Valid format check
Phone → 10-digit validation
Subject → Required selection
Message → Minimum 10 characters

Shows real-time error messages and success message on submit.
